End-to-end LLM serving platform built from scratch in Rust and Go. The Rust core implements a GPT-2 compatible transformer with sparse attention, a PagedAttention KV cache with fixed-size block allocator, and a custom time-series database backed by a lock-free skiplist memtable, LSM-tree with size-tiered compaction, B-tree indexing, and WAL for crash recovery. A Go HTTP backend handles request intake and GPU-aware scheduling, routing inference jobs to Rust workers via gRPC by KV cache locality. Deployed on Kubernetes with Prometheus metrics and Docker containerization.
